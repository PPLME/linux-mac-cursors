# linux-mac-cursors
guide of configuration for cursors style mac
## Installation Steps

1. **Download** the `mac-os-x-cursors.zip` file provided in this repository.
2. **Extract the file:** Right-click the zip file and select "Extract Here".
3. **Verify the folder content:** Ensure that inside your extracted folder there is an `index.theme` file. This is crucial for the system to recognize the cursor theme.
4. **Open your File Manager** and press **Ctrl + H** to reveal hidden folders.
5. **Locate the `.icons` folder:** If it doesn't exist, right-click to create a new folder and name it `.icons` (remember the dot at the beginning).
6. **Move the folder:** Cut the extracted "Mac OS X" folder and **paste it inside** your `~/.icons` directory.
7. **Apply the theme:**
   - Install GNOME Tweaks: `sudo apt install gnome-tweaks`
   - Open **Tweaks** and go to the **Appearance** tab.
   - Look at the **Cursor** dropdown menu and select **"Mac OS X"**.
   - Your cursor will change immediately. Enjoy your new look!
